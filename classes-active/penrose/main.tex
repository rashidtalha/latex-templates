\section{Introduction}
In this example document we describe how to use the \texttt{penrose} class. This is not a minimal working example, and might not cover all the features of this class. One should see the source code of this PDF for a more technical introduction. We have also used the \texttt{mpt-core} and \texttt{mpt-thm} packages written by the author to include brief samples of mathematical content. These are completely optional, and the user can use their own mathematical packages if needed.

\section{Metadata and The Title Block}
The class provides a flexible way to handle your document information. You can specify a subtitle and affiliation, which are automatically formatted in the title block. Note that the \verb|\title| command accepts an optional argument in square brackets; this is what appears in the page headers from page two onwards, preventing long titles from overflowing the header space.

Similarly, the \verb|\date| command is versatile. If you prefer a formal look, the standard command adds a ``Date:'' prefix. If you want a cleaner look, you can use the starred version \verb|\date*\{...\}| to print the date on its own.

Use \verb|\maketitle| to display the title block. Both the \verb|\maketitle| command and the \verb|abstract| environment are optional and should be used after \verb|\begin{document}|.

\section{Tables and Columns}
The class extends standard tables by providing custom column types through \verb|tabularx| and \verb|booktabs| packages. These are particularly useful when you need precise control over alignment and width. These are listed in table~(\ref{tab:tablename}). Additionally the standard column type \texttt{l, r, c, p\{<length>\}, X} defined by \verb|tabularx| are also available.
